A set of non-overlapping elementary catchments used to describe multiple inflows into a catchment aggregate and contributing to a joined outflow of the catchment aggregate. It primarily allows navigation to the 'highest' level system (total drainage basin) as typically used for reporting purposes. [HY_F]<br/><br/>
