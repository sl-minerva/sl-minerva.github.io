The following diagram is conceptual data model that is a simplified representation of the designed schema. The diagram shows the features and the relationships between them. The features and those relationships will also be explained here. The core features of the data model are elementary flowpath, hydro node, elementary catchment and Waterbody. These core features are aggregated to form larger interconnected features such as catchment aggregate, flowpath network and mainflowpath (mainstem). <br/><br/>
