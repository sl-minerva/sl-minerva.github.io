“A physiographic unit where hydrologic processes take place. This class denotes a physiographic unit, which is defined by a hydrologically determined outlet to which all waters flow …” [HY_F]. Catchments (and drainage basins) may have nested, hierarchical relationships. Following CHyF, NHNv2 allows for a strict definition of a catchment, in which case interior catchments are excluded and all areas explicitly drain to a common outlet; it also allows for a looser definition in which all fully contained interior catchments are considered as part of a large catchment; in this case it should technically be referred to as a catchment aggregate [CHyF]<br/><br/>"Une unité physiographique où les processus hydrologiques se passe. Cette classe désigne une unité physiographique, qui est définie par un exutoire hydrologiquement déterminé vers lequel toutes les eaux s'écoulent ..." [HY_F]. Les unités de drainages (et les bassins de drainage) peuvent avoir des relations hiérarchiques imbriquées. Comme pour CHyF, RHN permet une définition stricte d'un unité de drainage, auquel cas les unité de drainage intérieurs sont exclus et toutes les unités s'écoulent explicitement vers un exutoire commun ; il permet également une définition plus flexible dans laquelle tous les bassins versants intérieurs entièrement contenus sont considérés comme faisant partie d'un grand unité de drainage dans ce cas, il faut techniquement parler d’un ensemble d’unités de drainage [CHyF].<br/><br/>
