A hydro node is a network construct equivalent to a node or a vertex, and existing at each endpoint of an elementary flowpath. Seven types of hydro node are recognized: hydro nexus, bank nexus, flowpath nexus, water nexus, infrastructure (infrastructure junction), skeleton (skeleton junction), headwater and outlet. On reconnaît sept types de nœuds hydrologiques : nœud hydrologique, nœud de rive, nœud de filamentaire d’écoulement, nœud aquatique, nœud d’infrastructure, nœud de squelette, noeud supérieur ou inférieur.
